<b>What is OFI PHP Framework?</b>

<ul>
    <li> A PHP framework that takes the concept of the famous laravel and codeigniter framework </li>
    <li> A PHP Framework with more complect and fast
</ul>

OFI PHP Framework Ver 2.1

<b>Whats New?</b>
<ul>

</ul>
